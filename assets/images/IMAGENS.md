# Imagens do projeto White Medical (VITALIS)

As imagens do site ficam em **`assets/images/`**. Este arquivo documenta o mapeamento.

## Mapeamento (arquivo → uso no site)

| Arquivo | Uso | Páginas |
|---------|-----|---------|
| `laboratorio-medico.webp` | Hero / card de laboratório | design-system.html |
| `medica.webp` | Card da seção principal (médica) | index.html |
| `doctor.avif` | Avatar na seção de encaminhamento (equipe) | index.html |
| `doctor01.avif` | Avatar na seção de encaminhamento (equipe) | index.html |
| `waman.jpg` | Imagem da seção de depoimentos (98% satisfação) | index.html |
| `men.webp` | Avatar do depoimento (Michael Ross) | index.html |
| `waman01.webp` | Avatar do depoimento (Sarah Jay) | index.html |

## Caminho nas páginas

Use sempre o path: **`assets/images/nome-do-arquivo.ext`** (ex.: `assets/images/doctor.avif`).
